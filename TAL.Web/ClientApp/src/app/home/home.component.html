Hello TAL Team,

This is a single-page application, built with:

  • ASP.NET Core and C# for cross-platform server-side code
  • Angular and TypeScript for client-side code
  • PrimeNG for layout and styling

Here are some Assumptions that I made:

  • Date of Birth restriction to disallow future dates i.e. <= Today.
  • Age must be at least 1 year, otherwise the result will 0 regardless.
  • Sum Insured is limited to $10,000,000.00.
  • Development techniques used are:
    • Angular Components.
    • Angular Services
    • Angular Routing
    • Dependency Injection
    • Unit Tests
    • PrimeNG UI & Layout
    • Observables
    • Just to name a few...
